# Практическое задание курса Light Auto ML

**Соревнование —** [Regression of Used Car Prices](https://www.kaggle.com/competitions/playground-series-s4e9/overview)

**Постановка ML-задачи —** регрессия

**Бизнес-постановка задачи:** \
Спрогнозировать цену подержанных автомобилей на основе различных характеристик

**Цели проекта**
- Превзойти результаты бейзлайна на LAMA
- Продемонстрировать качественный код
- Использовать стандартные подходы к организации кода (например, Pipeline)
- Провести качественный EDA
- Предоставить подробное описание и обоснование гипотез

## Основные результаты

| Модель                            | Validation RMSE | Public Score | Private Score|
|-----------------------------------|-----------------|--------------|--------------|
| LightAutoML 1                     | 68512.86        | 72898.87     | 63930.99     |
| LightAutoML 2                     | 68041.68        | 72749.31     | 63669.16     |
| **LightGBM + Optuna**             | **67769.86**    | **72634.76** | **63442.93** |

## Основные этапы работы

- **EDA**: проанализировано распределение `price` (сильная асимметрия и выбросы), а также признаки и пропуски 
- **Разделение данных**: выполнен split на train/val для честной локальной проверки качества по RMSE
- **LAMA (baseline)**: обучены две конфигурации (TabularAutoML и TabularUtilizedAutoML) и выбрана более сильная по RMSE как baseline
- **Preprocessing**: обработаны пропуски, добавлены/извлечены признаки, выполнено кодирование категорий и подготовка данных к бустингу
- **LightGBM + Optuna**: проведён подбор гиперпараметров Optuna и обучение финального LightGBM, после чего сформирован файл submission

## Вывод

Подбор гиперпараметров LightGBM с помощью Optuna дал **лучшую валидационную метрику RMSE ≈ 67770**, что улучшило результат базовых моделей TabularAutoML и TabularUtilizedAutoML.\
На лидерборде сабмит `submission_LightGBM_Optuna.csv` показал **частный скор 63442.93** против ~63700 у бейзлайна LAMA, то есть тюнинг LightGBM позволил слегка побить AutoML-решение и получить более сильный финальный бейзлайн.
